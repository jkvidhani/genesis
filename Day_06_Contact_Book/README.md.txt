# 📒 Contact Book CLI

A command-line contact management system built in Python.

## 🚀 Features
- Add new contacts (name, phone, email)
- View all saved contacts
- Search contact by name
- Delete contact
- Data persists in a text file

## 🛠️ Built With
- Python 3.x
- File Handling
- OOP (Object Oriented Programming)
- Exception Handling

## ▶️ How to Run
```bash
python contact_book.py
```
